# Ultrasonic Parking Sensor

## Description
A smart parking assistant using the ESP32 and HC-SR04 ultrasonic sensor.

## Components
- ESP32
- HC-SR04
- Buzzer
- Red LED
- Yellow LED
- Green LED

## Features
- Green LED when the distance is safe.
- Yellow LED with intermittent buzzer for warning.
- Red LED with continuous buzzer for danger.
